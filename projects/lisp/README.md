# Lisp

# Setup
1. Install on common lisp compiler on your device `brew install clisp`

# Run code on clisp

- Interpreter - `clisp` and load file `(load "file.lisp")`
- Compile - `clip` and compile `(compile-file "file.lisp")` ~> file.fas next `(load "file.fas")`

# Exit from clisp

Just run `(quit)`
